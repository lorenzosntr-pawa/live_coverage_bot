**SportyBet API (from PROJECT.md):**
- Endpoint: `GET /liveOrPrematchEvents?sportId=sr:sport:1` (soccer)
- Response: `data[].events[]` with event details
- Provider IDs in event_id: `sr:match:{id}` for SportRadar, `sr:match:11111111{id}` for GeniusSports

**Event matching strategy:**
- SportyBet event IDs encode provider: `sr:match:12345` = SportRadar ID 12345
- Prefix `11111111` indicates GeniusSports: `sr:match:1111111112345` = GeniusSports ID 12345
- BetPawa `widgets[]` contains matching IDs for cross-reference
