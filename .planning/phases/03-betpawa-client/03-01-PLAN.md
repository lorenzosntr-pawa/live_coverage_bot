- Endpoint: `GET /events/lists/by-queries?q={JSON}`
- Query JSON: `{"queries":[{"query":{"eventType":"LIVE","categories":["2"]}}]}`
- Headers: `x-pawa-brand`, `x-pawa-language`, `devicetype`
- Response: `responses[0].responses[]` array of events
- Event fields: `id`, `name`, `widgets[]` (SPORTRADAR/GENIUSSPORTS IDs), `participants[]`, `results.display.minute`, `startTime`, `competition.{id,name}`
- Score: Extract from `participantPeriodResults[]` → find HOME/AWAY participant → `periodResults` where `period.slug = "FULL_TIME_EXCLUDING_OVERTIME"` → `result`
