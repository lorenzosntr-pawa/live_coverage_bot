Required query (working):
```python
query = {
    "queries": [
        {
            "query": {
                "eventType": "LIVE",
                "categories": ["2"],
                "zones": {},
            },
            "view": {
                "marketTypes": ["3743"],
            },
            "skip": 0,
            "take": 100,  # Use 100 to get all events, not just 20
            "sort": {
                "competitionPriority": "DESC",
            },
        }
    ]
}
```

Key changes:
1. Add `"zones": {}` inside the query object
2. Add `"view": {"marketTypes": ["3743"]}` at the query level
3. Add `"skip": 0` for pagination start
4. Add `"take": 100` to get more events (100 vs 20)
5. Add `"sort": {"competitionPriority": "DESC"}` for consistent ordering
