<task type="auto">
  <name>Task 2: Create configuration schema models</name>
  <files>src/live_coverage_bot/config/models.py</files>
  <action>
    Create Pydantic models for typed configuration:

    ```python
    class SportyBetConfig(BaseModel):
        base_url: str = "https://www.sportybet.com/api/ng"

    class BetPawaConfig(BaseModel):
        base_url: str = "https://www.betpawa.ng/api/sportsbook/v3"
        brand: str = "betpawa-nigeria"
        language: str = "en"
        device_type: str = "web"

    class SlackConfig(BaseModel):
        webhook_url: HttpUrl

    class Settings(BaseSettings):
        polling_interval_seconds: int = 30
        priority_leagues: list[str] = []  # Competition IDs to monitor
        sportybet: SportyBetConfig = SportyBetConfig()
        betpawa: BetPawaConfig = BetPawaConfig()
        slack: SlackConfig

        model_config = SettingsConfigDict(
            env_prefix="LCB_",
            env_nested_delimiter="__"
        )
    ```

    Use HttpUrl for webhook validation (catches malformed URLs early).
    Use SettingsConfigDict for environment variable overrides.
    Priority leagues as list[str] - competition IDs from API responses.
  </action>
  <verify>python -c "from live_coverage_bot.config.models import Settings" works</verify>
  <done>All config models importable, type hints complete, defaults sensible</done>
</task>
